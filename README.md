# Interactive Quiz App

This is an interactive quiz application built with HTML, CSS, and JavaScript. It features a questionnaire with random question selection, sound effects, and animated transitions.

## Features

- Random question selection from a pool of 30 questions
- Immediate feedback on answer selection
- Sound effects for correct and incorrect answers
- Animated transitions using CSS
- Responsive design suitable for both desktop and mobile devices

## Getting Started

### Prerequisites

- A modern web browser
- Basic understanding of HTML, CSS, and JavaScript
- (Optional) A local server for testing

### Installation

1. Clone the repository:
```
git clone https://github.com/yourusername/interactive-quiz-app.git
```

2. Navigate to the project directory:
```
cd interactive-quiz-app
```

3. Open `index.html` in your web browser.

## Usage

1. Click the "Start" button to begin the quiz.
2. Read each question carefully and select your answer.
3. Receive immediate feedback on your selection.
4. Click "Next" to proceed to the next question.
5. Use the hidden "Reset" button in the bottom right corner to start over.

## Customization

- To add or modify questions, edit the `questions` array in the JavaScript section of `index.html`.
- To change sound effects, replace the audio files and update the file paths in the `<audio>` tags.
- To modify the appearance, edit the CSS styles in the `<style>` section.
